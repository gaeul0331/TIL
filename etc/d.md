## 활용 방법

1. 이미지 저장 후 포토샵에서 열기.
2. ctrl + n 사용하여 원하는 썸네일 크기에 맞춰 새문서 열기.
3. 이미지를 복사하여 새문서에 붙여넣기.
4. ctrl + t
5. shift 누른 채로 마우스 움직이며 새문서 사이즈에 이미지 크기 맞추면 끝.
